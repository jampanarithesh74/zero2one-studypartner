# Z2O — ZERO2ONE Study Partner

> One Destination. Endless Possibilities.

ZERO2ONE (Z2O) is a student-built academic platform designed to simplify exam preparation by providing handwritten notes, PYQs, PDFs, Drive backups, and fast-access study resources for engineering students.

Built during exam season to solve a real student problem:
finding reliable study materials quickly without searching through multiple WhatsApp groups.

---

## 🚀 Features

- 📚 Handwritten Notes
- 📝 Previous Year Question Papers (PYQs)
- 📱 Mobile-Friendly UI
- ⚡ Fast PDF Preview System
- 🔗 Google Drive Backup Links
- 🧠 Subject-Code-Based Resource Organization
- 🌙 Clean Black + Orange UI
- 📂 Department-wise Resource Access
- 🔍 Scalable Subject-Centric Architecture (In Progress)

## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/z2o.git
```

Move into the project directory:

```bash
cd z2o
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=

VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```
